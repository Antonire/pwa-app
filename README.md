# PWA Приложение

## Функционал

### Главная страница:
- Добавление фото и текста
- Отображение постов в обратном хронологическом порядке

### Кебаб-меню:
- Кнопка перехода на главную страницу
- Кнопка создания нового раздела
- Список существующих разделов

### Разделы:
- Создание карточек с:
  - Названием локации
  - Картинкой
  - Описанием
  - Датой добавления
- Редактирование карточек
- Удаление карточек (с подтверждением)
- Карточки отображаются как новостная лента (новые выше)

## Установка и запуск

1. Создайте папку проекта со структурой, указанной выше
2. Создайте иконки разных размеров в папке `icons/`
3. Откройте `index.html` в браузере
4. Для PWA установки:
   - Chrome: нажмите "Установить" в адресной строке
   - Safari: нажмите "Поделиться" → "На экран «Домой»"

## Особенности

- Работает офлайн благодаря Service Worker
- Все данные хранятся в localStorage
- Адаптивный дизайн
- Подтверждение при удалении карточек
- Хронологическая сортировка карточек

## Технологии

- HTML5
- CSS3
- JavaScript (ES6+)
- Service Worker
- Web App Manifest
